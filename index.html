OpenLib

Full Stack Web Development

Objective:
    To provide a common platform for the campus community to buy/sell or borrow/share used semester books.
Methodology and Tech Stack:
    Designed the front-end using HTML, CSS, Javascript and implemented the back-end using Node.js, Express, and MongoDB for the database.
    Used javascript middleware to implement user authentication and Sendgrid API to send automated emails to the user.
    Implemented Mongoose search queries to search for the books in the database and their current availability status.
    Maintained a database record of book details, its current owner, and its original owner to keep the user updated.
Conclusion:
    Successfully created a web app for the easy transactions of books

SentilyZer

Natural Language Processing

Objective:
    To get a graphical representation of the public opinion using Natural Language processing (Sentiment Analysis) over a defined time interval.
Methodology and Tech Stack:
    Extracted the real-time tweets using Twint (a twitter scraping tool) and subjected the processed tweets to Textblob for its sentiment analysis and obtain its subjectivity and polarity
    Integrated the above functionalities into a web application. Implemented the backend in Django and the front end in HTML, CSS, and Bootstrap
    Included features like comparing opinions on two different topics, differentiating opinions by location(using Google Geocode API), and added a feedback system.
